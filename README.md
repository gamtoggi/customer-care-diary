# 고객 관리 수첩
개인사업자가 오프라인 수첩으로 관리해야 하는 고객정보, 고객구매정보, 재구매일정, 연락관리 등을
할 수 있는 웹 애플리케이션

## Available at
[CustomerNote.net](http://customernote.net)

## Skills
### Backend
Django
### Frontend
jQuery, Ajax, Bulma(css)
### DB
Mysql
### Server
Amazon EC2, Nginx, uWSGI

## 주요 기능
* 회원 가입/로그인
* 고객 인적사항, 구입 물품 등 저장/수정/삭제/검색
* 재구매 일정, 구매금액 순 등으로 고객 정렬기능
